Supported File Formats

CSV Files (Output or Stimulation Devices, e.g. Power Supply, VI Source, Resistor Cascade)

These files are .csv datalogs particularly exported from Keysight PathWave BenchVue software. Contains one channel per datalog file. For example, readings from a Keysight 34465A Digital Multimeter when connected to a VIO1304 card. Filename format:

<equipment_model>_<test_value>_<range_setting>_<channel_number>.csv

TXT Files (Input or Measuring Devices, e.g. Multimeter, Oscilloscope)

These files are in plaintext or ASCII format, copied or exported from Vector CANoe Trace window. Contains multiple channels per datalog file. Two possible formats: hierarchical (see VIO1008_m2V5_R5V.txt) and flat (VT2816A_60V_R60V.txt). Filename format:

<equipment_model>_<test_value>_<range_setting>.txt
